## 이 문서는 뭔가요

Claude Code에는 기본 기능 말고도 **더해서 쓸 수 있는 확장 기능**이 있습니다. 크게 세 종류입니다.

| 종류 | 한 줄 설명 |
|---|---|
| **플러그인** | 여러 도구를 세트로 묶어놓은 것 |
| **스킬** | 특정 작업 하나를 더 잘하게 해주는 능력 |
| **MCP 서버** | Claude를 노션, 슬랙 같은 다른 앱과 연결해주는 다리 |

전문 도구도 많이 섞여 있어서(법률, 금융, 마케팅 자동화 등) 전부 다 필요하지 않습니다. 아래는 **이 커뮤니티에서 만드는 종류의 사이트(1인 프로젝트, Astro/Next.js + Supabase + Vercel)** 에 실제로 도움이 될 만한 것만 골랐습니다.

> **먼저:** 이 커리큘럼을 무사히 마치는 게 우선입니다. 아래 것들은 사이트를 다 만들고 "더 손보고 싶다" 싶을 때 하나씩 시도해보세요.

## 설치는 어떻게 하나요 (개요)

| 종류 | 설치 방법 |
|---|---|
| 스킬 · 플러그인 | Claude Code 안에서 `/plugin` 명령으로 찾아 설치합니다 |
| MCP 서버 | `claude mcp add` 계열 명령으로 내 계정·앱과 연결합니다 |

> **정확한 명령어는 버전마다 바뀔 수 있습니다.** 아래 예시는 방향을 보여드리는 것이고, 실제로 칠 때는 Claude Code에서 `/help`를 쳐서 **지금 내 버전에 있는 명령어**를 확인하세요. (확인 필요)

설치가 끝났다고 자동으로 다 되는 건 아닙니다. **스킬·플러그인은 설치해두면 필요할 때 Claude가 알아서 씁니다.** 그래도 안 쓰는 것 같으면 아래처럼 원하는 걸 직접 말로 시키면 됩니다.

## 결과물 디자인을 더 다듬고 싶다면: DESIGN.md

`frontend-design` 스킬 말고, **완성된 브랜드의 디자인 시스템을 통째로 참고하는 방법**도 있습니다.

**DESIGN.md**는 유명 브랜드(Claude, Stripe, Notion, Apple 등 여러 곳)의 색상·타이포그래피·컴포넌트 규칙을 마크다운 파일 하나로 정리해둔 것입니다. 이 파일을 프로젝트에 넣고 "이런 느낌으로 만들어줘"라고 시키면, AI가 그 스타일을 참고해서 일관된 UI를 만들어줍니다.

```
npx getdesign@latest add claude
```

- 출처: [github.com/VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md)
- 이 명령어를 치면 프로젝트에 디자인 참고 파일이 추가됩니다.
- **아직 이 사이트(코딩 없이 코딩)에는 설치하지 않았습니다.** 새 패키지라 실제로 넣기 전에 먼저 확인이 필요합니다.
- 참가자가 본인 프로젝트에서 써보고 싶다면, 새 폴더에서 먼저 시험해보고 마음에 들 때만 실제 프로젝트에 적용하는 걸 추천합니다.

## 자주 나오는 질문

**Q. 이거 몰라도 사이트 완성할 수 있나요?**
네. 6주 커리큘럼은 `core/01 도구 6종`만으로 끝까지 갑니다. 이 문서는 다 만들고 난 다음의 선택지입니다.

**Q. 설치는 어떻게 하나요?**
스킬·플러그인은 Claude Code 안에서 `/plugin`, MCP는 `claude mcp add` 계열 명령으로 합니다. 정확한 구문은 `/help`로 확인하세요. 위 "설치는 어떻게 하나요" 절을 참고하세요.

**Q. 설치했는데 안 돼요.**
버전이나 명령어가 바뀌었을 수 있습니다. 각 도구의 공식 저장소·안내 문서를 다시 확인하세요.

**Q. 전문 도구(법률, 금융, 마케팅 자동화 등)는 왜 안 다루나요?**
이 커뮤니티의 목표(1인 웹사이트 제작)와 결이 달라서 뺐습니다. 필요하신 분은 각자 검색해서 찾아보세요.
