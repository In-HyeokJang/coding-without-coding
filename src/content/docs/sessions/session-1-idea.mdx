## 오늘 함께 뚫는 것 4개

### 1. Node.js — 사이트를 돌리는 엔진

**설치:** nodejs.org → LTS 버전 다운로드 → 계속 "다음"

**확인:**
```
node -v
```
`v20.11.0` 같은 숫자가 나오면 성공.

**안 되면:** 터미널을 껐다 다시 켜세요. 90%는 이걸로 해결됩니다.

---

### 2. VSCode — 코드를 들여다보는 창

**설치:** code.visualstudio.com → 다운로드 → 설치

**확인:** 프로그램이 열리고 왼쪽에 파일 목록 자리가 보이면 성공.

**알아둘 것:** 코드를 쓰라고 있는 게 아니라 **보라고** 있는 겁니다.

## 오늘의 하이라이트 · 첫 명령 던지기

세팅이 끝나면 이제 AI에게 처음으로 시킵니다.

터미널에서 `claude` 를 실행하고, 이렇게 쳐보세요.

```
Next.js 프로젝트를 새로 만들어줘.
TypeScript 없이, 가장 기본 설정으로.
폴더 이름은 my-site 로 해줘.
```

AI가 명령어를 자동으로 쳐 내려가면서 파일이 좌르르 생깁니다.

**여기서 잠깐.** `y` 를 누르기 전에 화면을 한 번 보세요. 뭘 하려는 건지 대충이라도.

## 자주 막히는 곳

| 증상 | 해결 |
|---|---|
| `node -v` 가 안 먹힘 | 터미널 껐다 켜기 |
| `npm install` 하다가 권한 에러 | 관리자 권한 터미널로 다시 실행 |
| Claude Code 로그인이 안 됨 | 브라우저에서 claude.ai 로그인 먼저 확인 |
| 명령어를 쳤는데 아무 반응 없음 | 지금 어느 폴더에 있는지 확인. 프로젝트 폴더 안이어야 함 |

**3분 넘게 막히면 손 드세요.** 혼자 헤매지 않는 게 오늘의 규칙입니다.
