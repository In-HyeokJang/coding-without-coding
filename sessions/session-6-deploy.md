# 6회차 · 배포

> **오늘의 목표:** 내 사이트를 세상에 공개하기
> **오늘 손에 남는 것:** 누구나 열 수 있는 URL + 카드 한 장

---

## 오늘 미리 알아두면 좋은 것

**시즌 마지막입니다.**

지금까지 `localhost:3000` 은 **내 노트북에서만** 열렸어요.
오늘 그게 **누구나 열 수 있는 주소**가 됩니다.

```
https://내프로젝트이름.vercel.app
```

그리고 **6명이 같은 시각에 동시에 배포합니다.**

---

## 오기 전에

- [ ] GitHub 로그인 상태
- [ ] Vercel 로그인 상태
- [ ] 어제 완성한 사이트가 있는 노트북

---

## 오늘 하는 것 4단계

### 1단계 · GitHub 푸시 + Vercel 연결 (30분)

어제 커밋한 코드를 GitHub에 최종 푸시합니다.

```
방금까지 작업한 걸 GitHub에 다시 올려줘.
```

그다음 **Vercel 화면에서 클릭으로** 진행합니다.

1. vercel.com 로그인
2. **Add New → Project**
3. 내 GitHub 저장소를 고르고 **Import**

대부분 클릭만 하면 됩니다. 여기까진 쉬워요.

---

### 2단계 · 환경 변수 등록 (30분) — 오늘의 진짜 벽

**이게 뭔가요**
4회차에 복사해둔 Supabase 키 2개를 **Vercel에도 등록하는 것**입니다.

**왜 필요한가**
지금까지 그 키는 내 노트북 `.env` 파일에만 있었어요.
그런데 사이트는 이제 **Vercel 서버에서 돌아갑니다.**

**그 서버도 열쇠가 있어야 창고에 접속하죠.**

**어디에**
Vercel 프로젝트 → **Settings → Environment Variables**

**뭘 넣나**
4회차에 복사한 Project URL, anon key

---

> ## ⚠️ 오늘 에러의 90%가 여기서 납니다
>
> **① 변수 이름 대소문자가 틀림**
> `NEXT_PUBLIC_SUPABASE_URL` 처럼 정확히 맞춰야 합니다.
>
> **② 키를 복사할 때 앞뒤 공백이 딸려옴**
> 붙여넣고 나서 커서를 맨 끝으로 옮겨 확인하세요.
>
> **이 두 개만 조심하면 됩니다.**

---

**환경 변수를 안 넣으면 어떻게 되나요?**

**배포는 됩니다.** 근데 사이트에 들어가면 **데이터가 안 나옵니다.**
빈 화면만 뜨거나 에러가 납니다.

**막혔을 때 프롬프트**
```
Vercel에 배포했는데 사이트에서 데이터가 안 나와.
로컬에서는 잘 됐어.
환경 변수 이름이 뭐로 되어 있어야 하는지 알려줘.
```

---

### 3단계 · 3, 2, 1 동시 배포

**6명 다 Deploy 버튼 앞에서 대기합니다.**

아직 안 된 사람이 있으면 **다 같이 기다립니다.** 타스가 뚫어줍니다.

그리고 카운트다운.

```
3 ... 2 ... 1 ...
```

**동시에 클릭.**

---

**그다음 1~3분.**

빌드 로그가 좌르르 흘러갑니다. 6명이 동시에 화면을 쳐다봅니다.

그리고 초록색으로 **`Ready`** 가 뜨면서 —

**주소가 생깁니다.**

---

### 4단계 · 서로 폰으로 접속

**폰을 꺼내세요.**

옆 사람의 URL을 자기 폰에 입력합니다.

**내가 만든 사이트가 남의 폰에서 열립니다.**

내 노트북이 아니라, 남의 손 안에 있는 기계에서. 6주 전엔 상상도 못 했던 일이에요.

---

## 마지막 · 카드

1회차에 쓰셨던 카드가 있습니다. **6주 동안 타스가 보관했어요.**

오늘 돌려드립니다.

### 1. 읽습니다 (3분)

6주 전 본인 글씨입니다.

> **"6주 뒤 내가 만들고 싶은 것"**

조용히 읽으세요. 아무 말 안 해도 됩니다.

### 2. 적습니다

카드 아래에, **방금 생긴 URL을 손으로 적으세요.**

```
[6주 전 내 글씨]
    나만의 카페 기록 사이트를 만들고 싶다

[오늘 내 글씨]
    https://my-cafe-log.vercel.app
```

**같은 사람 손글씨 두 개가 한 장에 있습니다.**

6주 전 "못 만들 거야" 라고 생각했던 사람과, 오늘 만들어낸 사람이 종이 한 장에서 만납니다.

### 3. 한 마디

한 사람씩 짧게.

---

## 오늘 손에 남는 것

- **라이브 URL** — 누구나 열 수 있는 내 사이트 주소
- **카드 한 장** — 6주 전 내 글씨와 오늘 내 URL이 나란히 적힌 종이

**두 번째 것을 집에 가져가세요.**

---

## 배포가 안 되면요?

**괜찮습니다. 오늘 안에 무조건 됩니다.**

- 카운트다운은 다 같이 합니다. **실패해도 같이 누릅니다**
- 폰으로 접속하는 시간에 타스가 옆에 붙습니다
- 그래도 안 되면 회차 끝나고 남아서 같이 뚫습니다
- **카드는 받습니다.** URL 칸만 비워두고, 뚫린 다음에 적으세요

**혼자 남겨지지 않습니다.**

---

## 시즌이 끝나고

**여러분 사이트는 계속 살아 있습니다.**

- 언제든 다듬을 수 있어요. Claude Code에게 시키면 됩니다
- 고치고 GitHub에 올리면 **Vercel이 자동으로 다시 배포합니다**
- URL은 그대로입니다

**6주 전에 없던 것이 지금 있습니다.**

이제 여러분은 **AI를 쓰기만 하던 사람**이 아니라, **AI로 만들어내는 사람**입니다.
