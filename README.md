## 이 폴더가 뭔가요

광명시 청년 6명이 6주 동안 각자 웹사이트 하나를 만들어 배포하는 커뮤니티, **코딩 없이 코딩**의 운영 문서입니다.

시즌 1을 위해 만들었지만, **시즌 2·3에도 그대로 씁니다.** 회차 소재가 바뀌지 않는 한 `core/`는 손댈 일이 거의 없고, `sessions/`도 도구 버전 정도만 바뀝니다.

## 업데이트 규칙

이 폴더는 살아 있는 문서입니다. 도구는 계속 바뀝니다.

### 언제 고치나

| 상황 | 어디를 고치나 |
|---|---|
| Claude Code 설치 명령어가 바뀜 | `core/01`, `sessions/1회차` |
| Supabase UI가 바뀜 | `sessions/4회차` |
| Vercel 배포 화면이 바뀜 | `sessions/6회차` |
| 더 좋은 프롬프트를 발견함 | `core/02` |
| 회차 소재를 바꿈 | `sessions/` 전체 + 기획서 |

### 어떻게 고치나

1. 파일을 고친다
2. `CHANGELOG.md`에 한 줄 남긴다 (날짜 · 뭘 · 왜)
3. 이 README 맨 위 버전과 날짜를 올린다

### 버전 번호

- **1.0 → 1.1** — 오타, 설명 보완, 프롬프트 추가
- **1.0 → 2.0** — 도구 교체, 회차 소재 변경, 스택 변경
